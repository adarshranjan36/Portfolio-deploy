# Adarsh Ranjan - Portfolio

A beautiful and modern portfolio website built with React showcasing both Software Developer and Software Tester roles.

## Tech Stack

- React.js (Vite)
- TailwindCSS
- Framer Motion (animations)
- Formspree (contact form)

## Features

- Dual profile showcase (Developer & Tester)
- Responsive design
- Modern UI with animations
- Contact form with Formspree integration
- Project showcase
- Skills visualization
- Dark/Light mode support
- Image loading states

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm run install:all
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Contact Form Setup

The contact form uses Formspree. To enable it:
1. Go to https://formspree.io/ and create a free account
2. Create a new form and get your Form ID
3. Update the form ID in `client/src/components/Contact.jsx` (line 27)
